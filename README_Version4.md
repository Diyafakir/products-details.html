# Natural Glow - Natural Beauty Powders Website

A beautiful, minimal website for a natural beauty powders brand with a green theme.

## 👋 About

**Founder:** Diya Asif Fakir

**Natural Glow** is dedicated to creating pure, organic beauty powders that celebrate your natural beauty.

## 📦 Products

1. **Face Powder** - Lightweight, natural finish powder for all skin types
2. **Hair Powder** - Volumizing powder that absorbs excess oil
3. **Body Powder** - Silky smooth botanical powder for everyday freshness

## 🎨 Design Features

- **Minimal & Clean Design** - Simple, elegant layout focused on the products
- **Natural Green Theme** - Eco-friendly color palette
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Smooth Animations** - Subtle transitions and hover effects
- **Accessible** - Semantic HTML and proper contrast ratios

## 📋 Sections

- **Navigation** - Sticky header with smooth scrolling
- **Hero Section** - Eye-catching introduction
- **Products** - Showcase all three powder products
- **About** - Brand story and statistics
- **Contact** - Get in touch with contact form
- **Footer** - Social media links and copyright

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS variables for easy customization)
- Vanilla JavaScript
- Font Awesome Icons

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize colors, text, and images as needed

## 📱 Responsive Breakpoints

- Desktop: 1200px and above
- Tablet: 768px to 1199px
- Mobile: Below 768px
